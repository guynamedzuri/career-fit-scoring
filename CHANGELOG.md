# Changelog

## 1.0.0 (2024-12-XX)

### 초기 릴리스

- 자격증 점수 계산 알고리즘 (10점 만점)
- 경력 점수 계산 알고리즘 (20점 만점)
- 학력 점수 계산 알고리즘 (20점 만점)
- 총점 계산 (가중 평균)
- 커리어넷 API 연동 (직종 검색, 학과 검색, 직종 상세 조회)
- Q-Net API 연동 (국가자격증 검색)
- 공인민간자격증 파싱 유틸리티
- 서버 사이드 지원 (Node.js)

### 프로젝트 전환

이 모듈은 원래 웹 기반 ATS(Applicant Tracking System)에서 사용되었으나, 
이제 Electron 기반 데스크톱 애플리케이션으로 전환되었습니다.

**주요 변경사항:**
- Electron 앱용으로 재구성
- DOCX 이력서 파싱 기능 추가 예정
- 자유 형식 경력 분석 기능 추가 예정
- 로컬 파일 기반 처리 지원

**향후 계획:**
- DOCX 파싱 모듈 개발
- 경력 정보 자동 매핑 기능
- Electron 앱 통합
