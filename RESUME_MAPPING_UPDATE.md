# 이력서 매핑 업데이트 내역

## 업데이트된 매핑 설정

### 테이블 1: 기본 인적사항
- (1,1): 한글이름 + 한문이름
- (1,5): 희망연봉
- (2,1): 영문이름
- (2,5): 증명사진 (이미지)
- (3,1): 생년월일
- (3,4): 이메일
- (4,1): 현재 주소지
- (5,1): 자택전화
- (6,1): 이동전화
- (8,1~5): 병역 관련 (5개 셀)

### 테이블 2: 학력사항
- row 2~7: 각 행이 하나의 학력사항
- cell 0: 입학년월
- cell 1: 졸업년월
- cell 2: 학교명
- cell 3: 전공명
- cell 4: 학점 (평균점수/총점수)
- cell 5: 학교 소재지
- cell 6: 졸업구분

### 테이블 3: 경력사항
- row 2~6: 각 행이 하나의 경력사항
- cell 0: 입사년월
- cell 1: 퇴사년월
- cell 2: 회사명
- cell 3: 근무부서
- cell 4: 직위
- cell 5: 연봉
- cell 6: 이직사유

### 테이블 4: 어학/자격/연수/수상
- **어학 (row 2~4, cell 0~2)**:
  - cell 0: 어학종류명
  - cell 1: 점수/등급
  - cell 2: 취득일자

- **자격증 (row 2~4, cell 3~5)**:
  - cell 3: 자격증 이름
  - cell 4: 등급/점수
  - cell 5: 발행기관

- **해외연수 (row 6~8, cell 0~2)**:
  - cell 0: 연수 국가
  - cell 1: 거주기간
  - cell 2: 거주목적

- **수상경력 (row 6~8, cell 3~5)**:
  - cell 3: 수상명
  - cell 4: 수상기관
  - cell 5: 수상내역

### 테이블 5: 자기소개서
- (1,1): 자기소개서 답안 1
- (3,1): 자기소개서 답안 2
- (5,1): 자기소개서 답안 3
- (7,1): 자기소개서 답안 4
