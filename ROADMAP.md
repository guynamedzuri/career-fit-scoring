## 주요 도전 과제

### 1. 자유 형식 경력 분석
- **문제**: 사용자가 어떤 형식으로 경력을 작성할지 예측 불가
- **해결 방안**:
  - 정규표현식 기반 패턴 매칭
  - 자연어 처리 (NLP) 활용
  - 커리어넷 API를 통한 유사 직종 검색
  - 사용자 확인/수정 기능 제공

### 2. 데이터 매핑 정확도
- **문제**: 이력서 형식이 다양하여 자동 추출이 어려움
- **해결 방안**:
  - 템플릿 제공 (사용자가 특정 형식으로 작성하도록 유도)
  - 사용자 확인 단계 필수
  - 매핑 결과 저장 (학습 데이터로 활용 가능)

### 3. 성능
- **문제**: 대량의 이력서 파일 처리
- **해결 방안**:
  - 배치 처리
  - 캐싱 (API 호출 결과)
  - 비동기 처리
